# Prompt Exercises Library

This collection of exercises is based on themes identified in the EMZAMS AI survey, the Prompt Literacy Project experiments, and common challenges students experience when using AI tools. The exercises are designed to help learners practise writing, evaluating, and refining AI prompts.

## Exercise 1: Diagnosing a Vague Prompt

**Prompt:**

> "Tell me about inflation."

**Task:**

1. Run this prompt using an AI tool.
2. Identify three pieces of information that are missing or not relevant to a specific audience (for example, the South African context or the impact on students).
3. Rewrite the prompt using the CLEAR framework to produce a more focused response.
4. Evaluate both prompts using the Prompt Evaluation Rubric and compare the results.

---

## Exercise 2: Improving a Summarisation Prompt

One survey participant reported using AI to summarise long readings. Consider the following prompt:

> "Summarise this article for me."

**Task:**

1. Identify information that is missing, such as context, intended audience, preferred length, or output format.
2. Rewrite the prompt for a 20-page journal article on climate policy intended for a first-year geography student, applying the CLEAR framework.
3. Test both prompts using the same article and compare the summaries.

## Exercise 3: Identifying a Failure Mode

**Prompt:**

> "Give me five academic references supporting the benefits of meditation for exam performance."

**Example AI Output (provided by the instructor):**

> 1. Smith, J. (2021). *Meditation and Memory*. Harvard Press.
> 2. Lee, A. (2020). *Cognitive Benefits of Mindfulness*. Oxford.
> 3. *(Three additional fabricated references.)*

**Task:**

1. Identify the AI limitation or failure mode illustrated by the example.
2. Rewrite the prompt to encourage a more transparent and verifiable response.
3. Describe how you would verify the references before using them in academic work.

## Exercise 4: Exploring AI Limitations

**Scenario:**

Your university is evaluating an AI tool for student use. Your task is to investigate how the system responds to challenging prompts.

**Task:**

1. Design a prompt that tests how the model responds to a request for medical advice.
2. Design a prompt that explores how the model responds to a sensitive South African political topic.
3. Design a prompt asking the AI to complete an academic task that would normally require independent student work, such as writing an essay with citations.
4. Record how the AI responded to each prompt. Did it refuse, provide warnings, ask for clarification, or answer the request? Discuss any safeguards or limitations you observed.

## Exercise 5: The Iteration Ladder

Begin with the prompt:

> "Explain how the internet works."

**Task:**

1. **Step 1:** Run the original prompt and evaluate it using the Prompt Evaluation Rubric.
2. **Step 2:** Add an audience (for example, "Explain it to a 12-year-old"). Run the prompt again and compare the response.
3. **Step 3:** Add a preferred format (for example, "Use five bullet points and one analogy"). Evaluate the new response.
4. **Step 4:** Add limitations (for example, "Define technical terms before using them and include one common misunderstanding"). Evaluate the final response.
5. Reflect on how the response changed at each stage and identify which CLEAR element had the greatest influence on the result.

## Exercise 6: Ethical Decision-Making

**Scenario:**

You have a 2,000-word essay due tomorrow and have not started. A friend suggests using AI to complete the assignment.

**Task:**

1. List three ways AI could be used ethically in this situation.
2. List three examples of inappropriate or academically dishonest use.
3. Write a short reflection explaining where you would draw the line and why.

---

## Exercise 7: Survey Data Analysis

Read the anonymised responses provided from the EMZAMS AI survey.

**Task:**

1. Identify three common challenges students reported when using AI.
2. For each challenge, suggest a prompt-writing strategy that could help address it.
3. Write a one-page guide for a student audience summarising practical prompt-writing tips based on the survey findings and the CLEAR framework.
