# Experiment: Ambiguity

## Objective

Examine how the level of detail in a prompt influences the relevance and focus of AI-generated responses.

## Prompt Progression

### Initial Prompt

> Tell me about Java.

### Revised Prompt

> Tell me about the programming language Java.

### Improved Prompt

> Explain the Java programming language to a beginner. Cover its main uses, include one code example, and provide a brief history. Write the response in three paragraphs.

### Final Prompt

> Explain the Java programming language to a high school student who has learned basic Python and wants to build Android apps. Use only concepts they would already know from Python, such as variables, loops, and functions. Include one short code example, explain why Java is important for Android development, describe one common beginner mistake, and write the response in three paragraphs. Define any technical terms before using them.

## Analysis

The initial prompt is open to multiple interpretations because the term *Java* may refer to the programming language, the Indonesian island, or coffee. As additional information is included, the intended meaning becomes clearer.

Each revision adds more detail by specifying:

* the subject (the Java programming language);
* the intended audience (a beginner, then a high school student with Python experience);
* the expected content (history, code example, Android relevance, beginner mistake); and
* the desired structure and level of explanation.

These additions make it easier for the AI to generate a response that is relevant to the user's needs and appropriate for the intended audience.

## Key Findings

This experiment illustrates how reducing ambiguity can improve the focus and usefulness of AI-generated responses. Providing context, identifying the audience, and including clear requirements generally leads to more targeted and consistent outputs.

## Takeaway

When a prompt can be interpreted in more than one way, providing additional context and clear requirements helps reduce ambiguity and improves the likelihood of receiving a relevant response.

## Survey Connection

Several participants in the EMZAMS AI survey reported receiving responses that were too broad or not closely aligned with what they wanted. This experiment demonstrates how adding context and specific instructions can help address that challenge.
